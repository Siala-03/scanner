# Servv - Restaurant Management System

A full-stack restaurant management application with real-time order tracking, inventory management, and staff performance analytics.

## Features

- **Customer Ordering**: Browse menu, add to cart, place orders
- **Kitchen Display**: Real-time order queue with status updates
- **Waiter Dashboard**: Manage tables, mark orders as served
- **Inventory Management**: Track stock levels, record movements
- **Supervisor Reports**: Revenue analytics, staff performance
- **Manager Tools**: Menu management, staff management, QR code generation

## Tech Stack

- **Frontend**: React + TypeScript + Vite
- **Backend**: Node.js + Express + PostgreSQL
- **Real-time**: Socket.io

## Getting Started

1. Run `npm install`
2. Run `npm run dev` (starts frontend)
3. Run `npm run dev:backend` (starts backend separately)
